**Project: Upgrade Tank Monitoring System – Fuel Terminal Cikampek**
Created By : Citia Engineering Teams

This project involves the upgrade and enhancement of the Tank Monitoring System (TMS) used at the Fuel Terminal Cikampek to improve operational visibility, alarm management, and monitoring reliability of fuel storage tanks.
The upgraded system integrates Automatic Tank Gauging (ATG) Honeywell Enraf instruments, gateway communication services, and a web-based monitoring platform to provide real-time information about tank conditions during receipt, storage, and fuel distribution operations.

The system supports operational monitoring for 12 fuel storage tanks, with enhanced alarm logic and real-time flow monitoring to assist operators in detecting abnormal conditions during terminal operations.
Key Features

**1.	Monitoring Upgrade for 12 Storage Tanks**
    Implementation of an upgraded monitoring system for twelve fuel storage tanks at the Fuel Terminal Cikampek, providing           improved visualization and operational data access.
**2.	ATG Enraf Data Acquisition**
    Tank level and product data are acquired from Honeywell Enraf Automatic Tank Gauging (ATG) devices through the CIU Gateway,      using the FD Interface service as the communication layer.
**3.	Custom Tank Level Alarm Configuration (HH, H, L, LL)**
    Implementation of configurable alarm thresholds to define safe operational limits for tank storage levels.
**4.	Real-Time Flowrate Monitoring**
    Addition of real-time flowrate visualization during fuel transfer operations to support better operational awareness.
**5.	Flowrate Stuck Detection Alarm (During Receipt Operations)**
    Alarm logic implemented to detect abnormal flow conditions such as flowrate stagnation during fuel receipt operations.
**6.	Idle Condition Level Monitoring Alarm**
    Alarm generation when tank level changes are detected during idle conditions, indicating possible leakage, measurement           anomalies, or unintended transfer.
**7.	Abnormal Level Increase Alarm During Distribution**
    Alarm triggered if tank level increases during product distribution operations, indicating possible operational anomalies.
**8.	Web-Based Monitoring Application**
    Development of a web-based application to enable centralized and flexible monitoring of tank parameters and alarm conditions.
**9.	Large Display Monitoring Screen (50-inch TV)**
    Installation of a dedicated large display screen for real-time monitoring visualization within the control room environment.
**10.	Data Integration from Witness Server**
    Integration of operational data sourced from the Witness server to support centralized monitoring and historical data            reference.
